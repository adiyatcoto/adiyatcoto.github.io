What it does
📡
Multi-Room Scraper
Scrapes TradingView chatrooms (Bitcoin, Forex, Crypto, Stocks, etc.) across date ranges using authenticated sessions. Parallel workers via ThreadPoolExecutor.
🧠
Weighted NLP Sentiment
Custom bull/bear/neutral lexicons with weighted scoring and negation detection. Classifies each message and computes net sentiment per user and per day.
📊
Fear & Greed Index
Composite Fear & Greed score (0–100) derived from bull ratio, mean net sentiment, and momentum velocity — displayed as an interactive gauge.
🚨
Anomaly Detection
Z-score based daily anomaly flagging — automatically surfaces PUMP SIGNAL and DUMP SIGNAL events when sentiment deviates significantly from the rolling mean.
🤖
AI Behavioral Profiling
Sends message samples to Gemini 2.5 Flash for deep behavioral analysis — trading personality type, emotional tendencies, strategy signals, and predictive insights.
👥
User Influence Map
Bubble scatter chart mapping each user by message count vs avg sentiment — identifying top bullish and bearish influencers in the room.
🕐
Hourly Heatmap
Plotly heatmap showing net sentiment by hour × day — revealing when crowds are most bullish or bearish throughout the trading session.
⬇️
Export: Excel & CSV
Full data export with raw message data + user summary sheet (messages, avg sentiment, bull/bear counts) — ready for further analysis.
